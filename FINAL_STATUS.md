## 🏗️ 技术架构

### 技术栈
- **Flutter**: 3.10.1+ (跨平台 UI 框架)
- **Dart**: 3.0+ (编程语言)
- **process_run**: 系统命令执行
- **shared_preferences**: 本地设置存储
- **path**: 文件路径处理

### 架构设计
```
UI Layer (TerminalScreen)
    ↓
Business Logic (Command Processor)  
    ↓
Data Layer (Settings & History Storage)
```

### 设计模式
- **状态管理**: StatefulWidget + setState
- **命令模式**: 命令解析和执行分离
- **策略模式**: 内置命令 vs 系统命令
- **观察者模式**: UI 状态更新
