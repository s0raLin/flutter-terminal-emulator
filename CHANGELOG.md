## [1.0.0] - 2024-01-20

### 添加
- 🎉 初始版本发布
- ✨ 基础终端模拟器功能
- 📁 文件系统操作命令
  - `ls` / `dir` - 列出目录内容
  - `cd` - 切换目录
  - `pwd` - 显示当前目录
  - `mkdir` - 创建目录
  - `touch` - 创建文件
  - `cat` - 显示文件内容
- 🔧 内置实用命令
  - `help` - 显示帮助信息
  - `clear` - 清空终端
  - `echo` - 输出文本
  - `history` - 显示命令历史
  - `settings` - 打开设置对话框
  - `exit` - 退出应用
- 💻 系统命令执行支持
- 📚 命令历史功能
  - 使用上下箭头键浏览历史
  - 历史命令存储和管理
- ⌨️ 键盘快捷键支持
  - `↑` / `↓` - 浏览命令历史
  - `Ctrl+C` - 中断当前输入
  - `Enter` - 执行命令
- 🎨 可自定义界面
  - 字体大小调节（10-24px）
  - 文字颜色选择
  - 背景颜色选择
  - 设置持久化存储
- 📱 跨平台支持
  - Android
  - iOS
  - Windows
  - macOS
  - Linux
- 🔄 自动滚动到最新输出
- 💾 用户设置本地存储
- 🎯 响应式界面设计

### 技术实现
- 使用 Flutter 3.10.1+ 开发
- 基于 `process_run` 包执行系统命令
- 使用 `shared_preferences` 存储用户设置
- 使用 `path` 包处理文件路径
- 实现了完整的状态管理
- 异步命令执行避免 UI 阻塞
- 虚拟滚动优化性能

## 版本说明

### 版本号格式
使用语义化版本号：`主版本号.次版本号.修订号`

- **主版本号**: 不兼容的 API 修改
- **次版本号**: 向下兼容的功能性新增
- **修订号**: 向下兼容的问题修正

### 变更类型
- `添加` - 新功能
- `修改` - 对现有功能的变更
- `弃用` - 即将移除的功能
- `移除` - 已移除的功能
- `修复` - 问题修复
- `安全` - 安全相关的修复
